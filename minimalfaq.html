FAQ

Q: What's this ".git" folder installed?

A: This is the maintainer (arcticsn0w)'s git configuration. This allows him to easily push MHB to GitHub. It's nothing to be worried about, and should not take up space. This should be deleted as of 5.1.2; if it's still there, upgrade to 5.1.2!

Q: Where can I request features?

A: Email ReddestDream via Cydia.

Q: Where do I report installation problems?

A: Email arcticsn0w via Cydia.

Q: Why does Chrome fail to load pages?

A: Chrome fails to load some pages when Google Ads don't load. Use a different browser.

Q: Why aren't ads blocked in Twitter?

A: Due to the way Twitter works, we can't block ads in it.

Q: Does this block ads in Alien Blue?

A: Yes!

Q: Does this block ads in YouTube?

A: No, and it never will. For YouTube, install mikoto from the repo http://cydia.angelxwind.net.

Q: Why is BytaFont's Font Browser broken?

A: It's a bug in BytaFont that we can't fix.

Q: Why won't SBSettings load?

A: SBSettings fails to load with a particular host blocked. To fix this, please navigate to /etc/hosts in iFile and delete the line "0.0.0.0 extra.thebigboss.org". Note that this will cause ads in BigBoss to return.

Q: Why is loading Twitch in the web browser broken?

A: Google Ad issues. Loading Twitch in the Twitch app is not broken.

Q: Is there a toggle for this?

A: Yes! Use Hosts File Flipswitch and your choice of CC tweak/Activator Action.

Any questions about this will be immediately directed to this page.
